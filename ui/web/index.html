What's assumed at the start (channels, common values, CRS)

Model: a static adversary and a fully synchronous authenticated broadcast channel available to all. So:

  • Participant set is known: everyone knows the ids and the published public keys (the G2 BLS keys and the G1 ElGamal keys).
  • Broadcast channel is open to all. Private channels are not needed: encrypted shares ride the public broadcast.
  • Each party privately holds its two secret keys (BLS signing and ElGamal).

The four Pedersen generators and the ElGamal base are derived by hash-to-curve (nothing-up-my-sleeve), so every party reproduces them with no trusted setup. z-layer aggregation tree (SCRAPE PVSS - binary tree gossip)

Each dealer's PVSS share for the secret z is verified at the leaves, then pairwise aggregated up the tree: ceil(log2 n) rounds, n-1 messages (vs O(n^2) for the Pedersen layer). Grayed-out levels are yet to be processed.
